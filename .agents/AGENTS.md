# Правила для асистента (Rules for Assistant)

- Виконуй усі редагування файлів, запуск скриптів, консольних команд та git push **безпосередньо без запиту підтвердження**, якщо контекст завдання однозначний.
- Не запускай інтерактивні опитування (інтерв'ю) та не використовуй інструмент `ask_question`, окрім випадків, коли це критично необхідно через серйозну двозначність вимог.
- (Do not ask for confirmation or interview the user; execute all file edits, replacements, terminal commands, and git pushes directly unless explicitly asked).
